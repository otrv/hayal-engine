# Hayal

Hayal is a library to build games. The name comes from the Turkish word "hayal" which translates into "dream". The goal is to have an unintrusive but batteries included game library. The vision is to provide composability of [sokol](https://github.com/floooh/sokol) with the power of [Love2D](https://github.com/love2d/love).

> [!CAUTION]
> This project is built for my personal usage and it is very early in development. As such it is heavily underdocumented and lacks a lot of basic features. If you want to try it out or extend it you would need to dive into the code.

## Build

Running `zig build --help` should give you the information needed to build the examples and the library.
